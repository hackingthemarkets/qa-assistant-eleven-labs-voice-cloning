## Setup

You need an Eleven Labs API Key and an OpenAI API Key to use this.

You also need a Python environment. 

Install dependencies with:

```
pip install -r requirements.txt
```

On OS X, install `ffmpeg` with:

```
brew install ffmpeg
```
